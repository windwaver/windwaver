GSview
------
.. _extraire une page d'un document pdf:

Extraction d'une page d'un document pdf
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
1) ouvrir avec GSview
2) Fichier -> Convert
3) choisir la bonne page et mettre Type à pdfwrite.
